# 购房税费完全指南

## 一、税费总览

### 买方需缴纳

| 税费项目 | 新房 | 二手房 |
|---------|------|--------|
| 契税 | ✓ | ✓ |
| 维修基金 | ✓ | × |
| 产权登记费 | ✓ | ✓ |
| 中介费 | × | ✓ |
| 印花税 | ✓ | ✓ |

### 卖方需缴纳（二手房）

| 税费项目 | 说明 |
|---------|------|
| 增值税 | 不满2年需缴纳 |
| 个人所得税 | 非满五唯一需缴纳 |

> 注：实际交易中，卖方税费常由买方承担，需在合同中明确。

## 二、契税详解

### 税率标准

| 情况 | 面积≤90㎡ | 面积>90㎡ |
|------|----------|----------|
| 首套房 | 1% | 1.5% |
| 二套房 | 1% | 2% |
| 三套及以上 | 3% | 3% |

### 计算示例

**案例1**: 首套房，100㎡，总价150万
- 契税 = 150万 × 1.5% = 2.25万

**案例2**: 二套房，80㎡，总价120万
- 契税 = 120万 × 1% = 1.2万

**案例3**: 三套房，120㎡，总价200万
- 契税 = 200万 × 3% = 6万

### 首套房认定

- 家庭为单位（夫妻+未成年子女）
- 以购房所在城市为范围
- 查询名下住房数量

## 三、增值税详解

### 征收标准

| 房产证年限 | 税率 |
|-----------|------|
| 不满2年 | 5%（差额或全额） |
| 满2年 | 免征 |

### 计算方式

**全额征收**:
```
增值税 = 成交价 ÷ (1+5%) × 5%
```

**差额征收**（部分城市）:
```
增值税 = (成交价 - 原购买价) ÷ (1+5%) × 5%
```

### 计算示例

房产证不满2年，成交价200万：
- 增值税 = 200万 ÷ 1.05 × 5% = 9.52万

## 四、个人所得税详解

### 征收标准

| 情况 | 税率 |
|------|------|
| 满五唯一 | 免征 |
| 其他情况 | 1%（核定）或 20%（差额） |

### 满五唯一条件

1. **满五**: 房产证满5年
2. **唯一**: 卖方家庭在该城市唯一住房

两个条件必须同时满足才能免征。

### 计算方式

**核定征收**（常用）:
```
个税 = 成交价 × 1%
```

**差额征收**:
```
个税 = (成交价 - 原购买价 - 合理费用) × 20%
```

### 计算示例

非满五唯一，成交价200万：
- 个税 = 200万 × 1% = 2万

## 五、其他费用

### 维修基金（新房）

| 建筑类型 | 标准 |
|---------|------|
| 多层住宅 | 50-80元/㎡ |
| 高层住宅 | 80-120元/㎡ |

100㎡高层住宅：维修基金约1万元

### 产权登记费

- 住宅：80元/套
- 非住宅：550元/套

### 中介费

- 一般为成交价的1%-2%
- 可以谈判
- 部分城市有指导价

### 评估费

- 贷款需要评估
- 一般为评估价的0.1%-0.5%
- 部分银行免费

### 印花税

- 住宅：暂免
- 非住宅：0.05%

## 六、税费计算实例

### 案例：购买二手房

**基本信息**:
- 成交价：150万
- 面积：100㎡
- 首套房
- 房产证满3年（满2不满5）
- 非唯一住房

**税费计算**:

| 项目 | 金额 | 说明 |
|------|------|------|
| 契税 | 2.25万 | 150万 × 1.5% |
| 增值税 | 0 | 满2年免征 |
| 个税 | 1.5万 | 150万 × 1% |
| 中介费 | 2.25万 | 150万 × 1.5% |
| 登记费 | 80元 | 固定 |
| **合计** | **约6万** | |

### 案例：购买新房

**基本信息**:
- 成交价：120万
- 面积：90㎡
- 首套房
- 高层住宅

**税费计算**:

| 项目 | 金额 | 说明 |
|------|------|------|
| 契税 | 1.2万 | 120万 × 1% |
| 维修基金 | 0.9万 | 90㎡ × 100元 |
| 登记费 | 80元 | 固定 |
| **合计** | **约2.1万** | |

## 七、省税技巧

### 合法节税方法

1. **选择满五唯一房源**: 免个税
2. **选择满2年房源**: 免增值税
3. **控制面积**: 90㎡以下契税更低
4. **首套房认定**: 确保首套资格

### 注意事项

- 不要做阴阳合同（违法）
- 税费约定要写入合同
- 保留所有缴税凭证
